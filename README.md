# flutter_mask

A mask application by **flutter** and **firebase**.

Use firebase Authentication and Cloud_FireStore

出發點是因為疫情的關係，導致大家購買口罩必須消耗大量的時間排隊，甚至輪到了扣照卻售完，所以想說透過這個應用可以讓大家先登記。
